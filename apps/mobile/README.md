# تطبيق Youmi للهاتف (React Native + Expo)

واجهة الهاتف لسوق Youmi بالاتجاه العربي (RTL).

## التشغيل

من جذر المشروع:

```bash
pnpm install
pnpm --filter @youmi/mobile start
```

ثم افتح التطبيق عبر Expo Go على الهاتف، أو محاكي Android/iOS.

## الإعداد

- رابط الخادم يُضبط في `app.json` تحت `extra.apiUrl`.
- لمحاكي Android استخدم `http://10.0.2.2:4000/api` (يشير إلى localhost الجهاز المضيف).
- لجهاز حقيقي استخدم عنوان IP جهازك على الشبكة المحلية.

## الشاشات الحالية

- **Home**: عرض المنتجات في شبكة.
- **Product**: تفاصيل المنتج.

## خطوات لاحقة (مقترحة)

- سلة الشراء والدفع، تسجيل الدخول، الإشعارات (Push)، والدردشة.
